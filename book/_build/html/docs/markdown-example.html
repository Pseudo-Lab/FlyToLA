Jupyter Book은 markdown 문서를 지원합니다.

아래와 같은 예시 코드를 입력하면 markdown 문법이 적용됩니다.

# This is an h1 tag
## This is an h2 tag
###### This is an h6 tag

*This text will be italic*
_This will also be italic_

**This text will be bold**
__This will also be bold__

_You **can** combine them_

* Item 1
* Item 2
  * Item 2a
  * Item 2b

1. Item 1
1. Item 2
1. Item 3
   1. Item 3a
   1. Item 3b

  2. Item 2
